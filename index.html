# IMEx data - A non-redundant set of physical molecular interaction data from a broad taxonomic range of organisms. - Expertly curated from direct submissions, peer-reviewed journals or pre-prints to a consistent high standard. - Available in standard formats MITAB or PSI-MI XML 3.0 - Provided by a network of participating major public domain databases.
